# 超声波



## 关于接线

具体开发手册和产品资料可见 [百度网盘链接](https://pan.baidu.com/s/1jQxyWd4KqIz7IbguvW-UoQ) 提取码: jqyp

超声波的线一共四根: 两根电源线; 剩下两根绿色 can H ; 红色 can L

如果超声波的数据成功经过设备发出, 对应 CAN 口的灯会亮

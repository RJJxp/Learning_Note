# Qt 设置

Qt 的轻量版和专业版, 前者1个g, 后者10个g

然而安装后者写类的时候, 括号无法自动补全, 应该是一种 bug

并没有找到解决方案, 只好安装轻量版

但是轻量版并没有优美的字体和合格的配色方案

记得当时先把专业版安装一遍, 通过截图, 再用软件取 rgb 的方式获得其配色

然后自己安装 source code pro 字体

它的配色如下:

|    字体     |  R      |  G      |  B      |
| :------: | :---: | :---: | :---: |
|   局部变量 |   62   |    46    |   100   |
|函数|58|103|124|
|虚拟|238|102|89|
